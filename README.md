# DealVault Platform Escrow

Modern escrow web application for secure buyer-seller transactions.

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- Google OAuth
- JWT

### Payments
- Razorpay

---

## Core Features

- Secure escrow lifecycle
- Buyer/seller workflows
- Deal management
- Payment locking
- Delivery submission
- Approval/release flow
- Dispute handling
- Real-time notifications
